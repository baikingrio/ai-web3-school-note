# Learning Plan

## Goal

以 AI × Web3 School 的课程为主线，用每天约 1 小时推进 Hackathon 项目方向的学习和 proof-of-work。

## Daily 1-Hour Routine

1. **10 分钟**：查看 WCB Learning 页面，确认今日课程 / 任务 / 打卡入口。
2. **20 分钟**：阅读 Handbook 或课程材料，记录关键概念与问题。
3. **20 分钟**：推进一个最小实验、任务或 Hackathon 项目模块。
4. **10 分钟**：更新 daily note，整理打卡草稿与下一步。

## Path Options

- **最小路径**：完成今日课程 + 写 daily note + 生成打卡草稿。
- **推荐路径**：完成今日课程 + 一个小实验 / 小 PR / 项目推进点。
- **挑战路径**：把今日知识直接转化为 Hackathon 项目功能、架构设计或 demo。

## Weekly Review

每周至少复盘一次：

- 本周完成了哪些 proof-of-work？
- 哪些内容能进入 Hackathon 项目？
- 哪些 Handbook feedback 值得整理提交？
- 下周最重要的 1–3 个推进点是什么？

This repository is public. Do **not** commit API keys, private keys, seed phrases, unpublished contact information, internal meeting links, personal data, or other secrets.

## Directory Structure

- `profile.md`: learner profile and learning preferences
- `learning-plan.md`: weekly / daily learning plan
- `daily/`: daily learning notes and check-in drafts
- `tasks/`: task notes and proof-of-work records
- `experiments/`: technical experiments and prototypes
- `handbook-feedback/`: feedback for Handbook pages
- `hackathon/`: Hackathon ideas, plans, and project logs
- `submissions/`: submitted links, screenshots, or public proof records — **[Week 1 PoW Pack](./submissions/week1-pow-pack.md)**
- `templates/`: reusable note templates

## Learning Principle

Lightweight first: every study session should produce a small visible artifact, not just passive reading.

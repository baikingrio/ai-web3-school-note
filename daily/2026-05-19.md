## 课程笔记

### 关键概念

- Hermes 可以通过不同消息网关接入日常沟通工具，让 Agent 更自然地进入学习和项目协作流程。
- 这次重点补齐了微信作为消息网关的配置思路；此前我只实际使用过 Telegram 接入。
- 消息网关的价值不只是“多一个入口”，而是让学习任务、路径管理、提醒和复盘更贴近日常使用场景。

### 方法 / 案例

- 回看「AI Agent 入门 —— Hermes 从 0 到 1」，了解配置微信消息网关的大致步骤和方法。
- 对比自己之前使用 Telegram 的经验，开始理解 Hermes 在多平台消息入口下的使用方式。
- 将 Hermes 视为长期学习助手，而不是一次性的问答工具：它可以帮助记录、拆解、追踪和复盘学习路径。

### 我的问题

- 微信消息网关和 Telegram 网关在稳定性、配置复杂度、使用场景上分别有什么优劣？
- 后续是否可以把每日学习打卡、Hackathon 任务拆解、资料整理和提醒都统一交给 Hermes 管理？

## 课程笔记

### 关键概念

- Agent Wallet 的核心不是“给 AI 一个私钥”，而是给 Agent 一个可验证、可限制、可撤销的行动边界。
- 对链上操作来说，权限边界要尽量由规则、合约、模块、Guard 和 Simulation 承担，而不是依赖 LLM 自觉。
- Hackathon MVP 应优先展示“能做什么”和“不能做什么”，尤其是拒绝路径。

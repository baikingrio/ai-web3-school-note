# AgentScoope Wallet — Hermes context

Project path: `experiments/agent-wallet/` (Sepolia testnet only).

## Architecture

- **Safe** holds USDC; **Agent EOA** signs `execTransactionWithRole` via Zodiac Roles Modifier.
- **App policy** (`config.json`): whitelist, maxPerTx, maxDaily (24h rolling from `logs/audit.jsonl`), expiresAt.
- **Human Check**: L0 auto, L1 `--confirm` / `HUMAN_CONFIRM=1`, L2 owner-only (reject), L3 policy/Roles reject.

## Key files

| File | Purpose |
|------|---------|
| `config.json` | Safe, agent, roles mod, policy (gitignored) |
| `.env` | `AGENT_PRIVATE_KEY`, `SEPOLIA_RPC_URL` (gitignored) |
| `src/pay.ts` | Single execution entry |
| `src/tools/run.ts` | Hermes-facing JSON tools |
| `hermes/SKILL.md` | Skill instructions |
